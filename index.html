Bind Manager

A browser-first key binding system for games and interactive applications with keyboard and gamepad support.

  • Rebind actions in a grouped modal with multiple slots and reset support.
  • Persist bindings in localStorage and subscribe to binding changes at runtime.
  • Capture keyboard and gamepad input, including profile-aware controller labels.
  • Show in-game hint overlays and open built-in input remap and controller test tools.

EventMergerJS

A library for coalescing repeated events into fewer updates while still preserving single incremental values. It is useful when you want to reduce network event load without losing important one-off changes.
